# Ambient Expense Agent

## Objective

Build an enterprise-grade expense approval workflow using Google ADK.

## Users

- Employees
- Managers
- Finance Team
- Compliance Team

## Features

- Expense submission
- Auto approval under $100
- PII redaction
- Prompt injection detection
- Risk assessment
- Human approval workflow
- Audit trail
- Evaluation framework

## Non Functional

- Secure
- Observable
- Testable
- Production Ready
